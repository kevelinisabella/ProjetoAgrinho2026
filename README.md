# ProjetoAgrinho2026
Este site foi desenvolvido como parte do Projeto Agrinho, com o objetivo de apresentar a importância e as características do agro no sudoeste do Paraná, destacando na agricultura, a sustentabilidade, a preservação ambiental e as tecnologias utilizadas no campo de forma criativa e interativa. 

O Projeto também serve como prática de programação WEB utilizanbdo linguagens HTML, CSS e JavaScript.

# Funcionalidades
- Layout moderno responsívo;
- Galeria de imagens do agro;
- Cards informativos;
- Navegaçao suave;
- Menu de acessibilidade
- Efeitos visuais;
- Quiz interativo;
